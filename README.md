# Smart Home Voice Control System

A responsive web application that allows users to control simulated smart home devices using voice commands. Built with PHP, Bootstrap, and JavaScript.

## Features

- Voice command interface using Web Speech API
- Simulated smart devices (lights, fans, door locks, entertainment systems)
- Responsive design with Bootstrap
- Multilingual support
- Command logging
- Customizable voice commands
- Accessible UI

## Requirements

- PHP 7.4 or higher
- Web server (Apache/Nginx)
- Modern web browser with Web Speech API support

## Installation

1. Clone this repository to your web server directory
2. Ensure your web server has PHP enabled
3. Access the application through your web browser

## Usage

1. Open the application in a modern web browser
2. Click the microphone button or use the keyboard shortcut (Ctrl + Space) to start voice input
3. Speak commands like:
   - "Turn on the living room lights"
   - "Set fan speed to medium"
   - "Lock the front door"
   - "Play music in the living room"

## Supported Languages

- English (default)
- Spanish
- French
- German

## Security Note

This is a simulation system and does not connect to real smart home devices. In a production environment, proper security measures should be implemented.
